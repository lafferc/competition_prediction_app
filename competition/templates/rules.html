Rules for {{TOURNAMENT.name}} (How to Play)

Predict the margin by which the home team will win.

Your prediction for each match is compared with the result. The difference is then added to your "score".

The player with the lowest "score" at the end of the competition wins.

Enter a positve number for a home win, a negitive number for an away win or 0 for a draw. You may choose decimal values for predicted margins.

{% if TOURNAMENT.draw_definition == "normal_time" %}

A match is considered a draw if both teams are level at the end of normal time. Extra/additional time is not counted. Shootouts are not counted.

{% elif TOURNAMENT.draw_definition == "extra_time" %}

If a match has extra/additional time, the match is considered a draw after all extra/additional time has been played. Shootouts are not counted.
